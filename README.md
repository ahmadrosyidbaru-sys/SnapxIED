# 📸 Snap Studio

**Bawa Studio Ke Layar Kamu.** Platform photobooth digital paling interaktif! Tanpa perlu install aplikasi, abadikan momen seru bareng temen tongkrongan pakai puluhan font, filter warna, dan template kekinian langsung dari browser.

![Snap Studio Preview](https://via.placeholder.com/800x400?text=Screenshot+Snap+Studio+Di+Sini)

---

## ✨ Fitur Utama

- **🌐 100% Berbasis Web:** Gak perlu proses instalasi yang ribet. Tinggal buka link, kasih izin kamera di browser, dan langsung jepret. Kompatibel di PC, Laptop, maupun Smartphone.
- **🎨 Kustomisasi Ekstrem:** Bebaskan kreativitas. Pilih layout (Grid atau Strip), atur warna *background*, ganti font nyentrik dari Google Fonts, dan pakai filter warna secara *real-time*.
- **🖨️ Cetak & Unduh HD:** Hasil foto diraster dalam resolusi tinggi. Siap buat di-upload ke *feeds* sosmed atau dicetak ke kertas foto asli tanpa *watermark*.
- **⚡ Super Ringan:** Mengandalkan teknologi WebRTC & HTML5 Canvas tanpa butuh *backend* server yang berat.

## 🚀 Versi & Evolusi

Proyek ini terus berkembang dari eksperimen sederhana jadi platform yang solid:
- **V.1 (Awal Mula):** Purwarupa dasar yang ngebuktiin browser bisa menggabungkan gambar layaknya mesin photobooth beneran. Simpel dan cepat.
- **V.2 (Era Kustomisasi):** Pembaruan UI masif dengan *color picker*, filter *real-time*, dan belasan pilihan font.
- **SPX (Edisi Tematik):** Edisi spesial dengan *framing* dan stiker khusus.
